# **App Name**: Maratona Fácil

## Core Features:

- Hero Section com Busca: Apresenta imagem impactante e barra de busca intuitiva para encontrar corridas por cidade, data e distância.
- Listagem de Corridas: Exibe corridas em cards responsivos com nome, data, local e distância, com link para detalhes.
- Detalhes da Corrida: Página detalhada com banner, descrição completa, opções de inscrição e botão de compra.
- Autenticação Simplificada: Formulários de login/cadastro acessíveis via modal, simulando o estado de login com localStorage.
- Carrinho de Compras (Mock): Simulação de carrinho de compras e checkout com revisão de itens e confirmação.
- Dashboard do Organizador (Mock): Painel simulado com métricas de inscrição e gerenciamento de corridas.
- Persuasão AI Tool: Generates messages that provide additional reasoning and help persuade runners to purchase specific tickets to events

## Style Guidelines:

- Fundo principal: Off-white (#F7F7F7) para clareza.
- Cor de destaque: Roxo Vibrante (#6B46C1) para botões e interações.
- Cor do texto: Preto ou cinza escuro para legibilidade.
- Fonte principal: 'Inter', sans-serif, para clareza e modernidade.
- Design mobile-first e responsivo com Tailwind CSS.
- Cantos arredondados e sombras suaves para profundidade.
- Transições suaves entre seções usando JavaScript para simular a navegação em um único arquivo HTML.
